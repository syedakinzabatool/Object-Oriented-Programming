# Object-Oriented-Programming

This repository contains simple C++ programs for learning and practicing Object-Oriented Programming (OOP) concepts.

## What is in this repo

- `Basics/`: beginner-level OOP and C++ programs (classes, constructors, destructors, pointers, static members, etc.).
- `Inheritance/`: examples of single, multiple, multilevel, hierarchical inheritance, and related problems.
- `Operator Overloading/`: programs that overload operators like increment, decrement, equality, and subscript.
- `Polymorphism/`: examples of function overloading, abstract classes, pure virtual functions, and compile-time/runtime behavior.
- `File Handeling/`: basic file handling example in C++.
- `Sem_Project/`: semester project code (game-related programs).

## Purpose

The goal of this repository is to keep OOP practice code in one place for study, revision, and lab tasks.

## Language

- C++
